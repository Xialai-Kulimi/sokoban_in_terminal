## Block
Example map
```
3 26
//////////////////////////
/01---------------------2/
//////////////////////////
```
1. `/` wall  
    牆壁  
2. `-` air  
    空氣  
3. `0` player  
    玩家  
    感覺可以在推動箱子的時候做一點動畫

4. `1` box  `[ ]`
    箱子  
5. `2` end  ` X `
    終點  
6. `3` placed `[X]`
7. `4` player_stand ` P `
